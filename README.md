# technical-test

## Instalation
untuk melakukan installasi telah dibuat file .sh untuk dapat melakukan installasi sekaligus menjalakan file secara development dengan perintah : 
```
$ source installation.sh
```

## Run Flask with Debug Mode
untuk melakukan running program flask menggunakan debug mode, dapat dilakukan dengan menjalankan `run.sh` dengan perintah :
```
$ source run.sh
```

## Run Flask in background
untuk melakukan running program flask pada background, telah disiapkan file `deploy.sh` dengan menjalankan perintah
```
$ source deploy.sh
```
